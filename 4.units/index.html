In CSS, rem stands for “root em”, a unit of measurement that represents the font size of the root element. This means that 1rem equals the font size of the html element, which for most browsers has a default value of 16px1. The rem unit is relative to the root—or the html—element. This means that if you set a font-size on the body element in your CSS, then all font-size declarations using rem units will be based on that value2.
